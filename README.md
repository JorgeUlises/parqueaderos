#Parqueaderos
Es el entorno de desarrollo para el sistema de parqueaderos.
```bash
#$ vagrant init centos/7
#Para iniciar la máquina
$ vagrant up
```
Se ejecuta la provisión:
```bash
$ vagrant provision
$ vagrant reload
$ vagrant ssh
```

¿MÁS FÁCIL?
-----------
```bash
$ ./install
```
